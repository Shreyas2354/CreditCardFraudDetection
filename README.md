# 💳 Credit Card Fraud Detection System

An AI-powered fraud detection web application developed using Machine Learning and Streamlit. The system analyzes transaction details in real time and predicts whether a transaction is legitimate or fraudulent.

---

## 🚀 Live Demo

🔗 https://creditcardfrauddetection12.streamlit.app/

---

## 📌 Project Overview

Online payment systems are highly vulnerable to fraudulent activities and unauthorized transactions. This project was developed to detect suspicious credit card transactions using Machine Learning techniques and real-time transaction analysis.

The application predicts fraud based on multiple transaction-related factors such as:
- Transaction Amount
- Merchant Category
- Foreign Transaction Status
- Device Trust Score
- Transaction Velocity
- Location Mismatch
- Transaction Time

The system provides:
- Real-time fraud prediction
- Fraud probability analysis
- Dynamic risk factor detection
- Intelligent recommendations
- Modern interactive dashboard
